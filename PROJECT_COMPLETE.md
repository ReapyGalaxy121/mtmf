## 🎓 What You Can Do Now

### As a User
```bash
# 1. Install
cargo install --path .

# 2. Initialize
mtmf init

# 3. Generate a key
mtmf key generate --alias my-key

# 4. Mint your first NFT
mtmf mint artwork.png

# 5. Check everything
mtmf doctor
```
